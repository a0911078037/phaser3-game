# phaser3-game
> 利用 phaser3 遊戲引擎，建立一個簡單網路遊戲  
> 遊戲介面皆為設計組成員繪製
## 操控方式
鍵盤上的上下左右對應人物的動作  
遊戲開始畫面有詳細的介紹。
